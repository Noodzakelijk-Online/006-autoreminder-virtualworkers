🍪 Clear VA Dashboard Cookies

⚠️ Important: This page must be opened from localhost:3001

To clear cookies for the VA Dashboard, you need to:

  1. Make sure the dev server is running: pnpm dev
  2. Open this file through the dev server at:
    http://localhost:3001/clear-cookies.html
  3. Click the "Clear Cookies" button below
  4. Click "Go to Dashboard" to test the login form
Ready to clear cookies. Click the button below.

Alternative Method (Manual)

If the button doesn't work, you can clear cookies manually:

  1. Press F12 to open DevTools
  2. Go to Application tab (Chrome) or Storage tab (Firefox)
  3. Find Cookies → http://localhost:3001
  4. Delete the manus-session cookie
  5. Refresh the page (F5)
